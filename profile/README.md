# 🍅 Open Tomato

**Open Tomato** is an open exploration of agentic development.

This organization gathers the outcomes of an ongoing research effort focused on how autonomous agents can be designed, coordinated, monitored and evolved in real-world systems. Instead of presenting a single “finished product,” Open Tomato offers a collection of experiments, tools, and building blocks that reflect different approaches to the same core problem: making agent systems more effective, adaptable, and composable.

Across the repositories, you’ll find:

* **Multi-agent platforms** that orchestrate parallel, headless workflows
* **Coordination layers** that track and evaluate agent behavior over time
* **Distributed learning models**, where agents develop individual “instincts” and successful patterns are promoted and shared
* **CLI tools, mobile interfaces, and libraries** designed to make these ideas usable in different environments
* **Modular components** that can be combined like building blocks to reduce complexity and context overhead. Great for those who really like only some stuff and trully dislike others. Make your own sauce.

A key idea behind Open Tomato is that agents shouldn’t just execute—they should **learn, diverge, converge, and improve collectively**. Some projects explore how independent agent threads can evolve distinct behaviors, while higher-level coordinators analyze outcomes and distill the most effective strategies into shared capabilities.

---

### 🧪 A research-first approach

Open Tomato is not a polished commercial product—and it’s not trying to be one.

Think of it like a **fresh tomato**: raw, versatile, and full of potential—but not a ready-made sauce.
If you’re looking for something canned, pre-seasoned, and immediately served… you might want a different kitchen.

Here, the focus is on **understanding the ingredients**, experimenting with recipes, and sharing what works (and what doesn’t).
